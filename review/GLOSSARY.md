# Kuyper Translation Glossary

> Dutch → English terminology for *Antirevolutionaire Staatkunde*. Add new terms as encountered during review.

## Core Theological Terms

| Dutch | English | Notes |
|-------|---------|-------|
| Souvereiniteit in eigen kring | Sphere Sovereignty | Central structural principle. Never "sphere sovereignty" (lowercase) in formal contexts. |
| Gemene Gratie | Common Grace | God's restraint of sin and bestowal of natural gifts upon the unregenerate. |
| Ordinantiën Gods | Ordinances of God | Never "rules" or "laws." Divine creation-structures. |
| De Overheid | The Magistrate | Not "The State." Literally "The Over-hood" or "Authority." |
| Antithese | The Antithesis | Fundamental spiritual cleavage between Christian and Revolutionary worldviews. |
| Organisch | Organic | Living, God-ordained natural growth (family, society, nation). |
| Mechanisch | Mechanical | Artificial, state-imposed structures. |
| Palingenesis | Regeneration | "Birth again" — applied to social order, not just individual soul. |
| Gemeene Gratie | Common Grace | Alternate spelling of Gemene Gratie. |
| Hervorming | Reformation | Not "reform" — carries specific theological weight. |
| Wedergeboorte | Regeneration / New Birth | Individual spiritual regeneration. |
| Uitverkiesing | Election | Divine election, cor ecclesiae (heart of the church). |
| Barmhartigheid | Mercy / Compassion | |
| Genade | Grace | Distinguish from Gemene Gratie (Common Grace). |
| Zondeval | The Fall | The Fall of man. |
| Schepping | Creation | |
| Verlossing | Redemption | |
| Verbond | Covenant | |

| Staatkunde | Politics / Political Science | Not "Constitutional Law" (that would be Staatsrecht). |
| Rechtsstaat | Rule of Law State / Constitutional State | |
| Volk | People / Nation | Distinguish from "nation" (natie). |
| Rijk | Realm / Kingdom | |
| Land | Land / Country | |

| Vrijmaking | Liberation / Emancipation | Specifically the 1886 church liberation. |

| Dutch | English | Notes |
|-------|---------|-------|
| Niet A, maar B | Not A, but B | Preserve Kuyper's structural parallelism. |
| Gelijk X, zo ook Y | Just as X, so also Y | Preserve structural parallelism. |
| Nachtcollege | Night School | The informal Reformed study circles. |
| Gezelschappen | Fellowships | The old Reformed fellowship circles. |
